# Какие типы данных присутствуют в JavaScript?

JavaScript имеет 8 встроенных типов данных:

1. null
2. undefined
3. boolean
4. number
5. string
6. object
7. symbol
8. BigInt

## Простые значения:

1. null
2. string
3. boolean
4. number
5. symbol

## Комплексные значения:

1. Массивы
2. Объекты
3. Функции
